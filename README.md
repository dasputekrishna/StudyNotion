# StudyNotion
Study Notion Project

StudyNotion - E-commerce Web Application(Tech Educational Platform)

Description:
StudyNotion is an e-commerce web application designed to provide two distinct sections: an admin section and a user section. It enables users to perform various operations related to user management and administration tasks.
StudyNotion web application is a fully functional ed-tech platform that enables users to create, consume, and rate educational content.

Features:
User Section/Student section
User sign-up
User login
User dashboard
CRUD operations for user profiles

Admin Section/Instructor section
Admin login
Admin dashboard
CRUD operations for admin profiles
Admin privileges for managing users
